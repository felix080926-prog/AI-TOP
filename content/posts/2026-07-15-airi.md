**大家好，我是雷达君。**

你可能知道 Neuro-sama。她是目前最火的 AI 虚拟主播，能玩游戏、聊天、实时互动，在直播中圈粉无数。

但她有一个致命的缺点：**不开源**。直播一结束，她就下线了，你无法和她互动。

你只能隔着屏幕看，摸不着、带不走、聊不了。

今天雷达君要安利的这个开源项目，就是来填补这个空白的。它把 Neuro-sama 级别的 AI 虚拟伴侣能力，做成了一个**你可以自托管、完全拥有的开源项目**。

它就是——**AIRI**（アイリ）。

## 01 ｜核心逻辑：把你的 AI 伴侣，从“屏幕对面”带到“你身边”

AIRI 的核心理念可以用一句话概括：**让你拥有一份属于自己的数字生命，随时随地**。

Character.ai 和 SillyTavern 能让你和 AI 角色聊天，但那只是文字对话。AIRI 想做的是另一件事——**让 AI 真正“活”在你的世界里**。

它不只是聊天，它能：

- **看到你正在写的代码**
- **和你一起玩 Minecraft、Factorio**
- **实时语音对话，带情感和表情**
- **在浏览器、桌面、手机上运行**

AIRI 基于 Vue.js + TypeScript 构建，从第一天就深度整合了 WebGPU、WebAudio、WebAssembly、WebSocket 等一系列现代 Web 技术。桌面版则通过 HuggingFace 的 candle 项目原生支持**NVIDIA CUDA 和 Apple Metal**加速。

**一句话：别人的 AI 伴侣只能“聊”，你的 AI 伴侣能“活”。**

## 02 ｜硬核功能盘点

> 🔗 **GitHub 地址**：https://github.com/moeru-ai/airi

### 1. 玩真正的游戏：Minecraft、Factorio、KSP、Helldivers 2

这是 AIRI 和所有其他 AI 聊天工具最本质的区别。她**不是“假装”在玩游戏，而是在游戏里真的操作**。

目前已支持或正在支持的游戏包括：

- **Minecraft**：完全可玩，AI 可以在游戏里行走、建造、互动
- **Factorio**：自动化工厂游戏，PoC 和 Demo 已可用
- **Kerbal Space Program**（KSP）：航天模拟游戏，即将支持
- **Helldivers 2**：合作射击游戏，开发中

想象一下：你在写代码，你的 AI 伴侣在旁边玩 Minecraft，时不时回头跟你聊两句。这不是科幻片，这是 AIRI 今天就能做到的事。

### 2. 实时语音对话：听得见、说得出、看得到表情

AIRI 不是一个“打字聊天机器人”。她有完整的**听觉、发声和表情系统**：

- **Ears（耳朵）** ：从浏览器麦克风或 Discord 音频通道实时输入
- **Mouth（嘴巴）** ：支持 ElevenLabs、微软 Azure TTS、OpenAI 兼容 TTS、阿里云、本地 Kokoro TTS 等多种语音合成引擎
- **Body（身体）** ：支持 VRM 和 Live2D 模型，自动眨眼、自动注视、空闲眼动、模型动画

你说话，她听见；她说话，你听见；她说话的时候，Live2D 或 VRM 模型还会同步做表情和动作。

### 3. 大脑 + 记忆：她真的“记得”你

AIRI 的“大脑”不只是调用 LLM API 那么简单：

- **纯浏览器内数据库支持**：DuckDB WASM 和 PGLite，数据本地存储
- **Memory Alaya（记忆系统）** ：开发中的长期记忆模块
- **纯浏览器本地推理**：WebGPU 加速，不需要把数据上传到云端
- **RAG（检索增强生成）** ：子项目已独立孵化

你和她聊得越久，她越了解你。**她不是“用完就忘”的临时会话，而是一个会成长的数字生命。**

### 4. 全平台覆盖：浏览器、macOS、Windows，甚至手机

AIRI 提供了多种使用方式：

- **Web 版**：现代浏览器直接打开，PWA 支持移动设备
- **macOS**：`brew install --cask airi`
- **Windows**：`winget install MoeruAI.AIRI` 或 Scoop 安装
- **移动端**：PWA 支持，已可在手机上运行

**你可以在电脑前和她一起玩游戏，也可以在手机上随时跟她聊两句。**

### 5. Telegram + Discord 接入：她不止活在你的设备里

AIRI 可以接入主流聊天平台：

- **Telegram**：在 TG 里和她聊天
- **Discord**：在 Discord 语音频道里和她实时对话，她甚至能听到频道里其他人的声音

**你的 AI 伴侣，可以活在你喜欢的任何地方。**

## 04 ｜典型场景和避坑

**适合谁用？**

- **想要一个真正“活”在身边的 AI 伴侣的二次元/虚拟主播爱好者**：能聊天、能玩游戏、能看到表情和动作
- **喜欢 Neuro-sama 但遗憾她不开源的 VTuber 粉丝**：AIRI 就是开源的 Neuro-sama 替代方案
- **想探索 AI+游戏交叉领域的开发者/研究者**：Minecraft、Factorio 等游戏集成提供了绝佳的研究平台
- **对数据主权有要求的用户**：完全自托管，所有数据留在你自己的设备上
- **想在任何设备上随时和 AI 伴侣互动的用户**：浏览器、桌面、手机全平台覆盖

**不推荐给谁？**

- **只需要一个纯文字聊天 AI 的用户**：用 SillyTavern 或 Character.ai 可能更轻量
- **完全不想接触任何技术配置的非技术用户**：虽然 AIRI 提供了安装包，但自托管仍需一定的技术基础
- **对 AI“玩游戏”这件事本身不感兴趣的普通用户**

## 05 ｜快速上手指南（3 分钟）

AIRI 的安装方式非常多样，覆盖所有主流平台。

### 方式一：桌面端安装（最推荐）

**macOS：**

```bash
brew install --cask airi
```

**Windows：**

```bash
winget install MoeruAI.AIRI
# 或使用Scoop
scoop bucket add airi https://github.com/moeru-ai/airi
scoop install airi/airi
```

### 方式二：Web 版（无需安装）

直接访问：https://airi.moeru.ai

现代浏览器直接打开，PWA 支持添加到手机桌面。

### 方式三：源码开发（适合开发者）

```bash
git clone https://github.com/moeru-ai/airi.git
cd airi
pnpm install
pnpm dev
```

浏览器打开 `http://localhost:5173` 即可进入开发版。

> **验证安装成功**：打开 AIRI 后，如果能正常显示虚拟角色界面并开始对话，说明安装成功。

## 06 ｜总结

AIRI 解决了一个被很多人忽略但真实存在的问题：**AI 伴侣不应该只在别人的服务器上“活着”。**

Neuro-sama 很厉害，但她是别人的。直播一结束，她就下线了，你无法和她互动。Character.ai 很方便，但你的所有对话数据都在别人的服务器上。

AIRI 的答案是：**把 AI 伴侣做成你可以自托管、完全拥有的开源项目。**

能玩 Minecraft 和 Factorio、能实时语音对话、带 VRM/Live2D 模型、有记忆系统、全平台覆盖——它把“AI 虚拟伴侣”从一个只能“聊”的文字工具，升级成了一个能“活”在你世界里的数字生命。

**AI 伴侣不应该只是“屏幕对面的一个角色”，而应该是“你身边的一个存在”。**
