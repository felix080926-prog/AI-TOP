**大家好，我是雷达君。**

祝大家端午安康！

过去两年，“用框架搭个 Agent”已经不难了。难的是让 Agent 在真实任务里**稳定地跑完一长串步骤**：会规划、会管理上下文、会调工具、会委派子任务、出错了还能被人接管。

更扎心的是，框架迭代太快。书一旦付印，版本就被时间钉住了。你按照书上的代码敲完，发现 API 早就变了，跑都跑不起来。

今天雷达君要推荐的，是一个完全不一样的解决方案。它不是一本书，而是一套**开源课程**——由 LangChain 官方认证大使出品，基于 LangChain/LangGraph 生态，从零构建生产级 AI Agent 的完整实战指南。

它就是——**《Deep Agents 实战》**。

## 01 ｜为什么是“开源课程”，而不是“书”？

这门课的出品人叫**沧海九粟**，LangChain 官方认证大使，B 站万粉 UP 主，此前已经出版过《LangChain 实战》和《LangGraph 实战》两本纸质书。

但到了第三本，主题是 Deep Agents——一个比前两本更新、迭代更猛的方向。他犹豫了很久，最后做了一个决定：**这一本，不做成书，做成一门开源课程**。

原因很简单。纸质书一旦付印，版本就被时间钉住了。框架更新了，书上的代码就跑不通了。而 Deep Agents 这个领域，“几乎每个月都在变”。

开源课程能做的事情，纸质书做不到：

- **跟得上版本**——框架更新，课程就跟着更新
- **真正可动手**——每章配可运行示例、配套视频与讲义 PDF
- **和大家一起长大**——每一条勘误、每一个 PR、每一次提问，都会沉淀回课程本身

**一句话：这不是一本“写完了就死了”的书，而是一个“活着”的知识体系。**

## 02 ｜这门课解决什么问题？

目前中文世界里，成体系、可动手、又对齐最新版本的 Deep Agents 实战教程还很少。

这门课想讲透一件事：**从 Agent Framework 到 Agent Harness**。

Framework 让你“搭一个 Agent”。Harness 让 Agent 在真实任务里**稳定地跑完一长串步骤**。

课程按“**认知 → 核心 → 进阶 → 实战**”循序展开，目前已发布**8 章正文 + 2 讲准备篇**，后续持续更新：

**准备篇**：用 AgentSeek CLI 快速搭建模板应用、安装开发辅助技能

**认知篇**：

- 第 1 章：从 Agent Framework 到 Agent Harness——Deep Agents 的诞生逻辑
- 第 2 章：快速上手——5 分钟构建你的第一个 Deep Agent

**核心篇**（这是课程的重头戏）：

- 第 3 章：虚拟文件系统——Deep Agents 的 Context Engineering 核心
- 第 4 章：任务规划与分解——让 Agent 学会拆解复杂任务
- 第 5 章：子 Agent 与上下文隔离——让 Agent 学会委派
- 第 6 章：异步子 Agent——让主 Agent 同时驱动多个子任务

**进阶篇**：

- 第 7 章：Skills——可复用的 Agent 能力包
- 第 8 章：长期记忆——让 Agent 拥有跨对话的记忆

后续还有 Human-in-the-Loop、沙箱执行等进阶内容，以及实战篇（流式前端、数据分析 Agent、生产部署）正在规划中。

> 配套资源也非常完整：每章可运行示例代码、讲义 PDF、B 站视频合集、小红书图文合集。你可以在 GitHub Pages 上直接访问课程网站。

## 03 ｜谁适合学？

这门课的目标人群非常清晰：

- **想系统学习 AI Agent 开发、不满足于“调通一个 demo”的开发者**
- **已经在用 LangChain/LangGraph，想进一步掌握 Deep Agents 工程实践的工程师**
- **零基础但有学习热情的同学**——跟着章节顺序走就行

## 04 ｜硬核配套：模型算力 + 开发工具

这门课在“降低动手门槛”这件事上，做得非常到位。

**模型算力支持**：课程与**硅基流动（SiliconFlow）** 合作，提供一站式大模型云服务平台。通过课程专属注册链接注册并完成实名认证，可获得**16 元全平台通用代金券**，足够跑通全部示例。平台还提供免费的文本模型与向量（Embedding）模型。

入门/简单任务用免费的`Qwen/Qwen2.5-7B-Instruct`即可跑通；复杂场景（任务规划、上下文总结、多子 Agent 编排）小模型往往无法稳定跑通，需改用 SOTA 模型，如`nex-agi/Nex-N2-Pro`（当前限免）或`Pro/zai-org/GLM-5.1`。

更贴心的是，作者会把平台返利**全额回馈给学员**，汇集成“实验配额补贴池”——跟着课程做实验时额度不够用，可以联系作者申请额外配额。

**开发工具**：课程基于**AgentSeek**工程化套件，帮助学员快速搭建开发环境。还推荐安装两个 AI 编码助手技能——LangChain 开发指南和 LangSmith Trace 调试，在开发过程中获得框架级的专业指导。

## 05 ｜社区共建：这不是一个人在战斗

这门课还有一个独特之处——**它是开源的，欢迎所有人参与共建**。

用 Issue 反馈具体问题（内容勘误、代码报错、改进建议、网站问题）；用 Discussions 进行开放性讨论（Q&A 问答、General 闲聊、Ideas 点子、Show and tell 晒作品）。

**有效反馈与 PR 贡献者会登上课程贡献者墙，并有机会获赠 LangChain 官方社区（中国）的实体礼品**。

## 06 ｜总结

回头看这门课，它解决的核心问题是：**AI Agent 领域“学得快、忘得更快”的知识获取困境**。
