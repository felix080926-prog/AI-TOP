如果想给视频配音，ElevenLabs 效果确实好，但每个月几十美元的订阅费让人肉疼，而且所有音频数据都要上传到云端。想用语音输入提高效率，WisprFlow 同样要付费，隐私问题一样悬在那里。

市面上最好的语音工具，输入和输出各占一头——ElevenLabs 管“说”，WisprFlow 管“听”。你要同时用这两样，就得付两份钱、信任两家公司、忍受两次数据上云。

如果有一个工具，能把这两样全包了，而且**完全本地运行、完全免费、完全开源**呢？

今天雷达君要安利的这个项目，就是这样一个答案。它把语音克隆、文本转语音、全局语音输入、AI Agent 语音交互全部塞进一个桌面应用里——你需要的所有语音能力，一个软件全搞定。

它就是——**Voicebox**。

## 01 ｜核心逻辑：把语音的“输入”和“输出”缝在一起

Voicebox 把自己定义为**本地优先的 AI 语音工作室**——一个免费开源的 ElevenLabs 和 WisprFlow 替代品。

ElevenLabs 做的是“输出”：文字转语音、声音克隆。WisprFlow 做的是“输入”：语音转文字、全局听写。两个方向是同一个硬币的两面，但市面上最好的工具偏偏各做各的。

Voicebox 做的事很简单：**把两面缝在一起**。

你用一个桌面应用，既能克隆声音生成语音（ElevenLabs 做的事），又能用全局热键在任何软件里语音输入（WisprFlow 做的事）。中间还加了一个本地小模型做润色和角色人格管理。

最关键的是——**所有模型、所有语音数据、所有录音，全部留在你本地机器上，永远不会离开你的硬盘**。没有月度订阅费，没有用量限制，没有“你的声音归谁”的隐私疑虑。

**一句话：输入输出全包，数据本地自持，不花一分钱。**

## 02 ｜硬核功能盘点

> 🔗 **GitHub 地址**：https://github.com/jamiepine/voicebox

**1. 7 款 TTS 引擎 + 23 种语言：按场景自由切换**

Voicebox 集成了目前开源社区最能打的 7 款 TTS 引擎：

- **Qwen3-TTS**：多语种主力，阿里巴巴出品，几秒音频即可实现几乎完美的零样本语音克隆
- **Qwen CustomVoice**：自然语言控制语音表达，你可以直接告诉它“用兴奋的语气说这句话”
- **LuxTTS、Chatterbox Multilingual、Chatterbox Turbo**：多语言覆盖，Turbo 版支持`[laugh]`、`[sigh]`、`[gasp]`等拟声标签
- **HumeAI TADA**：情感表达丰富
- **Kokoro**：内置 50+预设音色

支持的语言覆盖英语、阿拉伯语、日语、印地语、斯瓦希里语等**23 种**。你可以按场景灵活切换引擎：多语种用 Qwen，情感表达用 Chatterbox Turbo，预设音色用 Kokoro。

**2. 零样本语音克隆：几秒钟音频，复制你的声音**

这是 Voicebox 最核心的能力。你只需要提供**几秒钟的参考音频**，Voicebox 就能生成一个属于你自己的声音模型。

支持从音频文件创建声音档案，也可以直接在应用里录音。还支持**多样本组合**——把多个样本合并，获得更高质量的克隆效果。生成的语音可以无限长度，自动分块并用交叉淡入淡出拼接。所有档案可以导入导出，方便备份或分享。

**3. 全局语音输入：在任何软件里用嘴打字**

Voicebox 内置了基于 Whisper 的语音识别（STT）。你可以设置一个**全局热键**，在任何软件里按下就能开始语音输入。支持按住说话和切换两种模式。macOS 上还做了无障碍验证的自动粘贴，应用内每个文本框都自带麦克风按钮。

写邮件、写文档、聊微信——不用动手，说话就行。

**4. 故事编辑器：多轨时间线，做播客和对话**

Voicebox 内置了一个**多轨时间线编辑器**。你可以在一个项目里安排多个语音轨道，修剪和拆分音频片段，创建包含多个角色的对话。做播客、做有声书、做虚拟嘉宾对话——全在本地完成。

**5. 给 AI Agent 装上声音：MCP 协议一键接入**

这是 Voicebox 面向开发者最酷的功能。它内置了一个**MCP 服务器**，任何支持 MCP 协议的 AI Agent（Claude Code、Cursor、Cline 等）都可以通过一个工具调用`voicebox.speak`开口说话。

你克隆了自己的声音，然后让你的 AI Agent 用你的声音说话——这可能是你离“数字分身”最近的一次。

**6. 原生性能 + 全平台支持**

Voicebox 用**Tauri（Rust）** 构建，不是 Electron——内存占用更低、启动更快。macOS 上利用**MLX/Metal**加速，Apple Silicon 推理速度比同类快 4-5 倍。支持 macOS（Intel/Apple Silicon）、Windows（CUDA）、Linux、AMD ROCm、Intel Arc，甚至 Docker。

**7. REST API + 后期处理**

完整的 REST API 可以把语音合成能力集成到你自己的应用里。内置音高偏移、混响、延迟、合唱、压缩等后期效果链。所有生成内容都有历史记录，可按语音、文本、日期搜索和重新生成。

## 04 ｜典型场景和避坑

**适合谁用？**

- **需要语音克隆但不想付费订阅的内容创作者/播客制作者**：Voicebox 完全免费，无使用限制
- **对数据隐私敏感的团队或个人**：所有语音数据、模型权重全部留在本地，不上云端
- **想在 AI Agent（Claude Code、Cursor 等）中集成语音能力的开发者**：MCP 协议一键接入
- **需要多语言语音合成的国际化项目**：7 款引擎 + 23 种语言，按需切换

**不推荐给谁？**

- **只需要云服务一键部署、不想碰本地配置的用户**：Voicebox 需要下载模型和配置环境，有一定门槛
- **Linux 用户**：预编译二进制暂不可用，需要从源码构建
- **完全不想接触任何技术配置的纯内容创作者**：虽然 Voicebox 有桌面应用，但 TTS 引擎的下载和切换仍需一定的技术理解

## 05 ｜快速上手指南

Voicebox 的安装非常简单，提供了多平台支持：

**方式一：桌面应用（最推荐）**

- **macOS (Apple Silicon)**：[下载 DMG](https://voicebox.sh/download/mac-arm "下载 DMG")
- **macOS (Intel)**：[下载 DMG](https://voicebox.sh/download/mac-intel "下载 DMG")
- **Windows**：[下载 MSI](https://voicebox.sh/download/windows "下载 MSI")

下载安装后打开应用，在设置中下载你需要的 TTS 引擎和语音模型即可开始使用。

**方式二：Docker（适合服务器或开发者）**

```bash
docker compose up
```

**方式三：Linux（从源码构建）**

Linux 的预编译二进制暂不可用，可参考[官方 Linux 安装指南](https://voicebox.sh/linux-install "官方 Linux 安装指南")从源码构建。

> **验证安装成功**：打开 Voicebox，在语音生成界面输入一段文字，选择一个克隆声音或预设音色，点击生成。如果能正常输出音频文件，说明安装成功。

## 06 ｜总结

Voicebox 解决了一个很朴素但一直被忽视的问题：**语音工具不应该被拆成“输入”和“输出”两个阵营，更不应该把用户的数据和隐私当作收费的筹码。**

ElevenLabs 做输出，WisprFlow 做输入，两家各占一头，用户想要全栈体验就得付两份钱、信任两家公司。Voicebox 用一个本地优先的桌面应用，把两件事全包了——7 款 TTS 引擎、23 种语言、零样本语音克隆、全局语音输入、多轨故事编辑器、MCP 协议 Agent 语音——所有能力全部本地运行，所有数据全部留在你的硬盘上。

语音工具越来越贵、隐私越来越贵的时代，有一个完全开源、完全本地、完全免费的方案把整个语音 I/O 栈打包给你——这才是 Voicebox 真正的价值。
