## **01. 开发者共同的“数据孤岛”难题**

现在的 AI 智能体（Agent）很强，但它们通常被困在“小黑屋”里：

  * **API 昂贵且稀少：**  99% 的网站不提供官方 API，想让 AI 查个小红书、微博数据比登天还难。
  * **登录态噩梦：**  传统的爬虫根本无法处理复杂的登录校验、验证码和反爬。
  * **隐私焦虑：**  把 Cookie 交给云端 Agent？你真的放心吗？



**直到 bb-browser 出现，它彻底掀翻了桌子：既然网站不给接口，那就让 AI 直接用人类的界面！**

* * *

## **02. 什么是 bb-browser？你的浏览器就是 API**

> **GitHub 地址：**` https://github.com/epiral/bb-browser` 
> **核心定位：**  一个为 AI Agent 打造的 **CLI + MCP 服务器** 。它直接控制你的本地 Chrome，利用你已有的登录状态，将整个互联网“API 化”。

* * *

## **03. 核心杀手锏：为什么它是 Web Agent 的新标杆？**

  * **💥 自带登录态（Your Login is the API）：**  这是它最无敌的地方。你不需要提取 Cookie，不需要过验证码。只要你的浏览器里登录了账号，AI 就能直接调用！
  * **🔌 深度支持 MCP 协议：**  它内置了 Model Context Protocol。这意味着你可以直接在 **Claude Desktop**  中通过简单的对话，指挥它去查微博热搜、看 GitHub 趋势。
  * **🚀 社区驱动的“技能包”（bb-sites）：**  内置了 100+ 个平台的适配器（微博、小红书、知乎、京东、Boss 直聘...）。AI 不需要费劲解析 HTML，直接拿到的就是结构化的 JSON 数据。
  * **🔒 极致隐私（Local-first）：**  所有的通讯都在 `localhost` 完成。你的敏感数据从不离开你的机器，隐私安全拉满。
  * **📦 零配置安装：**  告别扩展程序！最新的 v0.8.0 版只需一行 `npm install -g bb-browser`，它能自动管理独立的 Chrome 实例，即装即用。



* * *

  * **全网情报官：**  “帮我汇总微博和小红书上关于‘DeepSeek’的所有热议话题，生成对比摘要。”
  * **极速调研员：**  “在雪球搜索某股票近期的热点讨论，并导出为报表。”
  * **求职加速器：**  “自动在 Boss 直聘上搜索北京地区的 Python 岗位，并整理符合条件的简历要求。”

## **05. 怎么使用？**

只需一行指令：
    
    
    npm install -g bb-browser  
    # 启动 MCP 服务器或直接命令行搜索  
    bb-browser site weibo/hot  
    

**就这么简单！**  接下来，你的 AI 就拥有了看遍全网的“上帝视角”。
